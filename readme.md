## Base commands

rustup doc

cargo new hello_world
cargo new my_lib --lib
cargo check
cargo build
cargo run
cargo build --release

update all dependencies:
cargo update

generate and open documentation of the project and its dependencies:
cargo doc --open

cargo test
cargo test --help
